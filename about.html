About Us

The Lit Lounge is a student-built online bookstore created for the CSC441 Web Application Development course. Our mission is to design an interactive, full-stack web application that allows users to browse and learn more about fiction books across various genres.

Project Overview

This site was developed to demonstrate our understanding of front-end and back-end technologies including HTML5, CSS, JavaScript, Node.js, MySQL, and MongoDB. The project features multiple pages, database-driven content, JavaScript interactivity, and form validation.
